Requirement: pandas 1.5 or lower

Usage: 

For causal rule mining:

      python CRM-discovery.py <file>, where file is a yaml textfile that lists line by line the input parameters, including the data CSV file. 

For repairing based on mined causal rules:

      python CRM-repair.py <file>, where file is a yaml textfile that lists line by line the input parameters, including the data CSV file.
